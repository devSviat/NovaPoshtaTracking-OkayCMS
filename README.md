# Модуль NovaPoshtaTracking для OkayCMS

Модуль дозволяє створювати та відстежувати експрес-накладні (ТТН) Нової Пошти безпосередньо з адмін-панелі магазину.  
Автоматично оновлює статуси накладних та дозволяє фільтрувати замовлення за статусом доставки.

## Можливості

- Створення експрес-накладних (ТТН) через API Нової Пошти.  
- Відстеження статусів накладних в реальному часі.  
- Автоматичне оновлення статусів накладних через cron (кожні 10 хвилин).  
- Фільтрація замовлень за статусом Нової Пошти в списку замовлень.  
- Збереження додаткової інформації про доставку (габарити, вага, тип вантажу тощо).

## Вимоги

- PHP 8.0 (може працювати на 7.4, але не тестувався)
- OkayCMS v4.5.2  
- API-ключ Нової Пошти  
- Модуль NovaposhtaCost (для роботи з даними доставки)

## Встановлення

1. Скачайте останню версію модуля з [останнього релізу на GitHub](https://github.com/devSviat/NovaPoshtaTracking-OkayCMS/releases/latest).  
2. Розпакуйте архів на сервері в каталог: `{OkayCMS_root}/Okay/Modules/`  

## Важливо

**Рекомендується оновити бібліотеку `giggsey/libphonenumber-for-php` до останньої версії.**  
Для деяких телефонних кодів (наприклад, 077) може виникати помилка "User phone is invalid" при створенні накладної, якщо використовується застаріла версія бібліотеки. Для оновлення виконайте команду:

```bash
composer update giggsey/libphonenumber-for-php
```
